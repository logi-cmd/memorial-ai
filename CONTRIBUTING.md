# Contributing to Memorial AI

Thank you for your interest in contributing to Memorial AI.

## Development Setup

1. Clone the repo: `git clone https://github.com/logi-cmd/memorial-ai.git`
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env` and fill in your API keys
4. Run the dev server: `npm run dev`

## Pull Request Process

1. Create a feature branch from `main`
2. Make your changes
3. Ensure `npm run build` passes with zero errors
4. Submit a PR with a clear description of the change

## Code Style

- TypeScript strict mode
- Use existing component patterns as reference
- Keep changes small and focused

## Reporting Issues

Please open a GitHub Issue with:
- A clear description of the bug or feature request
- Steps to reproduce (for bugs)
- Expected vs actual behavior
